# Support

Carta Arcanum is community-maintained software for Carta Arcanum campaign
planning and operations.

## Where To Get Help

- Use [GitHub Issues](https://github.com/aditaa/carta/issues) for bugs,
  feature requests, and documentation gaps.
- Include the app version or Git commit, relevant setup details, and steps to
  reproduce when reporting a bug.
- For security issues, follow [SECURITY.md](SECURITY.md) instead of opening a
  public issue.

## Before Filing An Issue

Please check:

- The [README](README.md) for quickstart commands.
- The [install guide](docs/INSTALL.md) for Linux, WSL, MySQL, installer, and
  upgrade notes.
- The [admin guide](docs/ADMIN.md) for settings, users, health checks, and
  maintenance workflows.
